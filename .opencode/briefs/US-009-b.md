## 2. Objetivo desta demanda
#### 2.1. Resultado funcional alvo (desta subtask)
Template Thymeleaf `perfil/adolescente-editar.html` renderiza a tela `/perfil` com: avatar block (inicial do apelido ou foto), campos nickname/birthDate/email(readonly), save bar sticky que aparece só quando dirty, modal de bloqueio por idade com CTA único. Alpine.js gerencia dirty state (`markDirty()` em `@input`) e toggle da save bar. CSS aplica tokens do design system para avatar-block, save-bar, modal. i18n messages para microcopy pt-BR.

#### 2.2. Surface comportamental coberta
- **Avatar block:** mostra inicial do apelido (ou foto se existir), apelido em display xl, e-mail muted
- **Dirty state:** `markDirty()` em `@input` de nickname/birthDate; save bar aparece quando `dirty === true`
- **Save bar sticky:** botão "Salvar alterações" (primário) + "Descartar" (ghost); só visível quando dirty
- **Modal bloqueio por idade:** abre ao tentar salvar com birthDate fora de 13–17; CTA único "Entendi, manter como estava" reverte birthDate
- **E-mail readonly:** input com `readonly` + ícone cadeado + helper

#### 2.3. Constraints do produto/demanda + constraints do AGENTS.md
- **Stack travada:** SIM/NÃO — Alpine.js já incluído no base.html; CSS puro (sem Tailwind classes novas, seguir tokens existentes)
- **Compliance LGPD:** SIM — banner de sucesso reafirma "Teu progresso continua igual."

## 4. Observações factuais (gotchas)
- O UX spec (§3.8) define que o modal de bloqueio tem **um único CTA** ("Entendi, manter como estava") — não dois botões. Escape no modal reverte e fecha (recomendação do spec §9).
- A save bar usa `sticky; bottom: 0` com `linear-gradient(180deg, rgba(247, 244, 241, 0) 0%, --color-bg 40%)` para fade — seguir token exato do spec (§3.6).
- O avatar block usa `--font-display` (Bricolage Grotesque) para o apelido e `--text-3xl` (32px) para a inicial no avatar.
- Alpine.js já está incluído via CDN no base.html — não precisa de dependência extra.
- O spec UX (§4) define microcopy literal em pt-BR — copiar exatamente como está, sem traduzir ou adaptar.
